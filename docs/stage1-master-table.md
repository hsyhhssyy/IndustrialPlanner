## 备注

- 多产物配方在 Stage1 仅兼容，不参与生产结算。
- 首批业务闭环为：`originium_ore -> originium_powder`。
- Phase1 建筑视觉为粗边框矩形+端口箭头+中心中文名（名称不旋转）。
- 取货口为 `3x1`，Phase1 仅校验“长边贴边”；方向字段可在数据层按需显式配置。
- Phase1 tick 执行顺序已冻结为“先推进再结算检查”，后续若调整需同步更新验收与追踪矩阵。
