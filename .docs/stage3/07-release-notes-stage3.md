# Stage3 发布说明（草案）

## 版本信息

- 阶段：Stage3
- 状态：进行中
- 首次发布日期：待定

## 面向用户的变化

1. 新增电力统计：耗电、发电、净功率，即时/10 分钟/1 小时视图。
2. 缺电行为调整：缺电不再直接停机，未接网仍不能运行。
3. 抽水泵效率可配置并影响实际输送。
4. 反应池维持既有“5 共享槽位”口径（已验收）。
5. 修复反应池槽位分配残留逻辑，运行时统一按 5 共享槽位分配。

## 升级注意事项

- 不引入反应池槽位结构迁移。
- 已废弃需求不会在本阶段上线。

## 已知限制

- 自动布局与全局优化不在本阶段范围内。
- 复杂流体物理特性（压力/温度）不在本阶段范围内。

## 发布后观察指标

- 电力窗口统计稳定性
- 抽水泵吞吐与效率一致性
- 反应池回归稳定性
- Stage2 冻结主流程回归率

## 已废弃需求说明

- 管道等效容量（`2 × 连通长度`）与单液体锁定/排空切换。
- 瓶装液体“隐藏组合 + 专用选择器 + 单展示多展开”。
