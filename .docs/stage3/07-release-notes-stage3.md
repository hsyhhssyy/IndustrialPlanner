# Stage3 发布说明（草案）

## 版本信息

- 阶段：Stage3
- 状态：进行中
- 首次发布日期：待定

## 面向用户的变化

1. 新增电力统计：耗电、发电、净功率，即时/10 分钟/1 小时视图。
2. 缺电行为调整：缺电不再直接停机，未接网仍不能运行。
3. 管道液体规则增强：单液体锁定、排空后切换、容量按管长计算。
4. 抽水泵效率可配置并影响实际输送。
5. 瓶装液体改为专用选择器，通用列表更简洁。
6. 反应池输入槽位升级为 3 固体 + 2 液体。

## 升级注意事项

- 旧版本反应池配置将执行迁移。
- 若看到液体阻塞提示，需先排空当前系统再切换液体。

## 已知限制

- 自动布局与全局优化不在本阶段范围内。
- 复杂流体物理特性（压力/温度）不在本阶段范围内。

## 发布后观察指标

- 电力窗口统计稳定性
- 流体阻塞与解锁正确率
- 反应池迁移成功率
- Stage2 冻结主流程回归率
