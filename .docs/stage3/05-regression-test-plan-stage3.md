# Stage3 回归测试计划

## 1. 测试目标

- 证明 Stage3 新能力可用。
- 证明 Stage2 冻结能力无高优先级回归。

## 2. 测试矩阵

### 2.1 电力模型

1. 设备功率字段读取与展示正确。
2. 即时净功率计算正确。
3. 10 分钟均值在 600 标准秒窗口内正确滚动。
4. 1 小时均值在 3600 标准秒窗口内正确滚动。
5. 缺电时设备继续生产（不降速不停机）。
6. 未接网设备仍不可运行。

### 2.2 抽水泵

1. 抽水泵效率配置加载成功。
2. 输出吞吐与效率一致。
3. 外圈放置规则与效率逻辑不冲突。

### 2.3 反应池（回归基线）

1. 反应池维持“5 共享槽位”口径。
2. 并行 lane 运行正常。
3. 路由与出料无回归。

### 2.4 已废弃项（不测）

- 管道等效容量（`2 × 连通长度`）与单液体锁定/排空切换。
- 瓶装液体“隐藏组合 + 专用选择器 + 单展示多展开”。

## 3. 验收门槛

- Stage3 新增项：P0/P1 缺陷为 0。
- Stage2 冻结主流程：P0/P1 回归为 0。
- 关键链路（电力/抽水泵/反应池）全部通过。
