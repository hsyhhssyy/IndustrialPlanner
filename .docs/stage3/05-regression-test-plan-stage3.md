# Stage3 回归测试计划

## 1. 测试目标

- 证明 Stage3 新能力可用。
- 证明 Stage2 冻结能力无高优先级回归。

## 2. 测试矩阵

### 2.1 电力模型

1. 设备功率字段读取与展示正确。
2. 即时净功率计算正确。
3. 10 分钟均值在 600 标准秒窗口内正确滚动。
4. 1 小时均值在 3600 标准秒窗口内正确滚动。
5. 缺电时设备继续生产（不降速不停机）。
6. 未接网设备仍不可运行。

### 2.2 流体容量与传染

1. 管道容量等于 `2 × 连通长度`。
2. 首次注入后系统锁定液体类型。
3. 异液注入被阻塞并给出提示。
4. 排空后允许切换到新液体。
5. 桥接器双通道隔离不互相污染。

### 2.3 抽水泵

1. 抽水泵效率配置加载成功。
2. 输出吞吐与效率一致。
3. 外圈放置规则与效率逻辑不冲突。

### 2.4 瓶装液体

1. 通用列表不出现瓶装组合项。
2. 专用选择器可选择瓶型与液体。
3. 瓶型无关配方展示单项。
4. 执行层展开后产线平衡正确。
5. 输出返瓶瓶型守恒。

### 2.5 反应池

1. 3 固体 + 2 液体槽位结构生效。
2. 并行 lane 运行正常。
3. 路由与出料无回归。

## 3. 验收门槛

- Stage3 新增项：P0/P1 缺陷为 0。
- Stage2 冻结主流程：P0/P1 回归为 0。
- 关键链路（电力/流体/反应池）全部通过。
