# Stage3 风险与回滚

## 1. 主要风险

### R1 电力语义切换风险（高）

- 描述：从“缺电停机”切换为“缺电不断产”可能影响既有平衡认知。
- 缓解：
  - 在 UI 明示当前功率状态。
  - 保留接网前置规则，避免无网运行误解。

### R2 管道传染规则风险（高）

- 描述：液体锁定后阻塞行为可能被误判为 bug。
- 缓解：
  - 增加阻塞提示文案。
  - 提供排空状态可观测信息。

### R3 反应池结构迁移风险（高）

- 描述：旧数据到 3+2 槽位映射可能失败或错配。
- 缓解：
  - 启动时迁移校验。
  - 无法映射项进入显式告警并允许人工修正。

### R4 瓶装液体隐藏策略风险（中）

- 描述：用户可能在通用列表找不到目标物品。
- 缓解：
  - 引导入口到专用选择器。
  - Wiki 提供瓶型/液体映射说明。

## 2. 回滚策略

- 触发条件：任一 P0，或关键链路 P1 未能在窗口内关闭。
- 回滚动作：
  1. 回退至 Stage2 冻结版本。
  2. 禁用 Stage3 新增持久化键读取。
  3. 重跑 Stage2 回归矩阵。
- 恢复条件：
  - 根因修复并通过 Stage3 + Stage2 双回归。
