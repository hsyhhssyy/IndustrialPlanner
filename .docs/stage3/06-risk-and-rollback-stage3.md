# Stage3 风险与回滚

## 1. 主要风险

### R1 电力语义切换风险（高）

- 描述：从“缺电停机”切换为“缺电不断产”可能影响既有平衡认知。
- 缓解：
  - 在 UI 明示当前功率状态。
  - 保留接网前置规则，避免无网运行误解。

### R2 抽水泵效率配置风险（中）

- 描述：效率参数配置错误会造成吞吐异常。
- 缓解：
  - 对配置值做范围校验与默认回退。
  - 增加吞吐对账用例。

### R3 文档口径漂移风险（中）

- 描述：已废弃需求或错误口径（反应池 3+2）在文档中回流。
- 缓解：
  - 统一以 Stage3 文档为单一口径。
  - PR 检查项加入“废弃项未回流”核对。

## 2. 回滚策略

- 触发条件：任一 P0，或关键链路 P1 未能在窗口内关闭。
- 回滚动作：
  1. 回退至 Stage2 冻结版本。
  2. 禁用 Stage3 新增持久化键读取。
  3. 重跑 Stage2 回归矩阵。
- 恢复条件：
  - 根因修复并通过 Stage3 + Stage2 双回归。

## 3. 废弃项说明

- 以下需求已废弃，不纳入风险与回滚评估：
  - 管道等效容量（`2 × 连通长度`）与单液体锁定/排空切换。
  - 瓶装液体“隐藏组合 + 专用选择器 + 单展示多展开”。
