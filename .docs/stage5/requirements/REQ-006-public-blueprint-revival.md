## 背景与目标

Stage5 需要提供一个可复用的起死回生机公共蓝图，用于演示、复用与验证蓝图库主流程。

## 当前口径

1. 蓝图必须纳入公共蓝图库索引。
2. 蓝图必须可浏览、可加载、可放置。
3. 若蓝图需要配套说明，应在蓝图文档或元数据中补齐。

## 当前状态视图

- 当前状态：`done`
- 当前目标：按 Stage5 发版结果归档公共蓝图交付。
- 当前已知进展：公共蓝图资产已接入 `public/blueprints/index.json`，运行时已具备公共蓝图索引拉取、蓝图内容同步、左侧公共蓝图列表显示、描述字段展示以及后续武装 / 放置链路；当前发布版本中已包含“一键去世机7.1”等公共蓝图样例。
- 下一步动作：
	1. Stage5 内无后续动作；若后续新增公共蓝图资产或分类规则，在下一阶段继续维护。

## 功能拆解

1. 准备或确认起死回生机蓝图资产。
2. 将其接入公共蓝图库索引。
3. 验证浏览、加载与放置链路。
4. 必要时补充展示名称、描述或分类信息。

## 验收标准

1. 蓝图索引中可见。
2. 蓝图可被成功加载。
3. 蓝图可被成功放置。
4. 不出现明显的索引丢失、加载失败或放置异常。
