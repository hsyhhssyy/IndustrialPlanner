## 背景与目标

Stage5 需要提供一个可复用的起死回生机公共蓝图，用于演示、复用与验证蓝图库主流程。

## 当前口径

1. 蓝图必须纳入公共蓝图库索引。
2. 蓝图必须可浏览、可加载、可放置。
3. 若蓝图需要配套说明，应在蓝图文档或元数据中补齐。

## 功能拆解

1. 准备或确认起死回生机蓝图资产。
2. 将其接入公共蓝图库索引。
3. 验证浏览、加载与放置链路。
4. 必要时补充展示名称、描述或分类信息。

## 验收标准

1. 蓝图索引中可见。
2. 蓝图可被成功加载。
3. 蓝图可被成功放置。
4. 不出现明显的索引丢失、加载失败或放置异常。
