# 逻辑元转发B站用文案

网页版《集成工业》产线仿真系统更新了 1.1.0 Beta 版本
-新增「来自PV盯帧的1.1版本超时空配方」支持
-支持自定义物流顺序

这次版本主要是为 1.1 新内容预铺路，也补上了一些更适合复杂设施调试的功能。
如果想提前规划新版本产线，可以试试看～

不过也要提醒一下：
超时空配方目前主要依据 PV 内容整理，和正式版如果有出入，请以游戏实装为准。
另外，自定义物流顺序可以做出部分游戏内未必能原样实现的效果，复杂设施最好还是进游戏再实际验证一次。

欢迎转发扩散 & 反馈问题～
