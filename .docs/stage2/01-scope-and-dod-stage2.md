# Stage2 范围与完成定义（冻结版）

## 1. In Scope（冻结）

- 产线规划器：
  - 顶栏入口、浮窗、状态持久化。
  - 多目标输入与聚合。
  - 地区切换与地区过滤。
  - 多配方手动切换重算。
  - 条件互转配方启用（蓝铁块/蓝铁粉末）。
  - 农业闭环平衡修正。
- 武陵地图流体基础：
  - 管道铺设与编辑。
  - 管道分流器/汇流器/桥接器。
  - 流体端口设备与流体配方接入。
- 放置约束：
  - 抽水泵仅外圈放置。
  - 外圈设施按标签规则放置。
  - 武陵建筑数量上限拦截。
- 反应池与水培：
  - 反应池两配方并行、输出路由配置。
  - 水培设备独立建模与接入。
- 交互治理：
  - Toast 降噪与关键告警保留。

## 2. Out of Scope（迁移至 Stage3）

- 电力功率模型（`powerDemand/powerSupply`）及净功率计算。
- 电力统计窗口：即时值、10 分钟、1 小时。
- 电力不足不停机/不降速语义改造。
- 管道等效缓存池容量（`2 × 连通长度`）。
- 管道单液体锁定/排空切换。
- 抽水泵输出效率配置化。
- 瓶装液体“隐藏组合 + 专用选择器 + 单展示多展开”。
- 反应池“3 固体 + 2 液体独立缓存位”模型。

## 3. DoD（冻结）

1. 规划器可完成多目标输入与结果输出。
2. 地区切换可驱动候选项过滤与重算。
3. 多配方可切换，切换后结果变化可见。
4. 蓝铁互转配方按条件启用。
5. 农业闭环不作为异常循环提示。
6. 武陵流体设施可放置并形成有效链路。
7. 固液运输边界校验生效。
8. 抽水泵外圈限制生效，内圈放置失败。
9. 武陵建筑上限拦截生效。
10. 反应池可配置并在仿真内运行。
11. 水培设备可在规划与仿真中使用。
12. Stage1 主流程无阻断回归。

## 4. 验收补充说明

- 未落地条目由 Stage3 文档承接并单独验收。
