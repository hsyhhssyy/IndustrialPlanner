# Stage2 发布说明（冻结版）

## 1. 版本信息

- 版本：Stage2-Frozen-2026-02-26
- 日期：2026-02-26

## 2. 本次确认已发布内容

### 2.1 规划器

- 新增规划器浮窗与多目标规划。
- 新增地区切换与地区过滤。
- 新增配方切换重算。
- 新增条件互转配方与农业闭环修正。

### 2.2 武陵流体

- 支持管道与管道组件（分流/汇流/桥接）。
- 支持流体设备端口与流体配方。
- 支持管道颜色化流体展示（无流体盒子动画）。

### 2.3 放置与规则

- 抽水泵外圈放置限制。
- 武陵建筑上限拦截。
- 固液运输介质分离。

### 2.4 特殊设备

- 反应池并行配方与输出路由。
- 水培设备独立建模接入。

### 2.5 体验治理

- Toast 降噪落地。
