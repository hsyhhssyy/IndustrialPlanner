# Stage2 变更总览与目标（冻结版）

## 1. 文档目的

- 记录 Stage2 按当前代码已落地能力。
- 明确 Stage2 与 Stage3 的边界，避免后续口径混淆。

## 2. 版本信息

- 冻结版本：Stage2-Frozen-2026-02-26
- 维护人：项目组

## 3. Stage2 已交付变更（相对 Stage1）

### 3.1 规划器能力

- 新增产线规划器入口与浮窗。
- 支持多目标吞吐输入与汇总计算。
- 支持地区切换（四号谷地/武陵）并过滤候选物品与配方。
- 支持多配方手动切换并触发重算。
- 支持蓝铁块/蓝铁粉末互转配方按条件启用。
- 支持农业闭环（种植机/采种机/水培）平衡修正。

### 3.2 地区与流体能力

- 武陵地区开放流体设施与流体配方。
- 管道、管道分流器、管道汇流器、管道桥接器可放置并参与仿真。
- 管道与传送带可共格，且与普通生产设备重叠受限。
- 固液运输边界生效：液体仅管道，固体仅传送带。

### 3.3 放置规则与设备能力

- 武陵扩展区放置规则生效（含抽水泵仅外圈）。
- 武陵特定建筑数量上限生效（如天有烘炉上限）。
- 反应池支持多配方配置、并行推进与液体/固体输出口定向。
- 水培设备作为独立设备接入。

### 3.4 观测与交互治理

- 保留 Stage1 仿真统计主链路（1 分钟窗口）。
- Toast 完成降噪治理（高频操作静默、失败/异常保留提示）。

## 4. 冻结差异说明（相对旧版 Stage2 草案）

- 反应池当前口径：5 共享槽位（非 3+2 独立槽位）。
- 抽水泵当前口径：支持多液体可选输出（非仅清水）。
- 电力当前口径：仍以 `requiresPower` 判定可运行，不含完整功率统计模型。

## 5. 已迁移至 Stage3 的需求

- 完整电力模型（`powerDemand/powerSupply`、净电力、10 分钟/1 小时窗口）。
- 电力不足不停机/不降速的运行语义。
- 管道等效缓存池容量模型（`2 × 连通长度`）。
- 管道单液体锁定与排空切换规则。
- 抽水泵效率配置化吞吐模型。
- 瓶装液体隐藏项与专用选择器完整交互。
- 反应池 3 固体 + 2 液体独立缓存模型。
